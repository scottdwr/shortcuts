Mobile WebAPP Configurator

Turn any website into an app on your homescreen. (JavaScript courtesy of @atnbueno)

Author: @scottdwr

Utilities

New version 2.1 DOES NOT require Toolbox Pro.

Creates a mobile profile configuration file which you email to yourself using an ‘iOS Mail’ mailbox.

Excellent way to have a WebAPP isolated from the rest of Safari. This way you don’t have to worry about losing your WebAPP data if you need to erase and reset Safari’s history.

Or, if you just want to log into an online service with different accounts at the same time.
