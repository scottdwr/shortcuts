Shortcut Source Tool

View, convert, save source in xml or json format, or edit and export source to Shortcuts

Author: @gluebyte

Development Tools

This is a developer tool with which you can:

  • Select a shortcut or open a source file (from .txt, .json, .xml, .plist or .shortcut) to:
  • Quick Look in xml or json format,
  • Convert from/to xml and json,
  • Save or share in xml or json,
  • Optionally edit source then export back to Shortcuts.
  • Convert and save source in Tinycut-style text format.
  • Useful for overview of structure and variable flow.
  • In addition to actions, it shows you (if conditions), “menu titles” and where variables (including named magic variables) are created (→) and consumed (←).

Enjoy. :)
