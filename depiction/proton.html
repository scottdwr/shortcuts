Proton

A simple yet powerful syntax highlighter for multiple programming languages

Author: @alombi

Development Tools

Welcome to Proton!

Proton is an useful developer tool that lets you to highlight your code's syntax in a spark. It's fast and easy, as it should be.



A clean and fast UI for focusing on what really matters. Just open Proton and see your code.

Share Sheet and automatic language detection

Just select Proton from your file's Share Sheet and you're done. Proton also automatically detects the language of your file using your file's extension, but you can still select the language manually if you would like.

Multiple themes

Proton features multiple themes, so that your code will always look awesome.
